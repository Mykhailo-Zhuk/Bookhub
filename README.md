# Bookhub — Платформа каталогу книг із відеооглядами (OPE-139)

**Linear:** OPE-139  
**Репозиторій:** https://github.com/Mykhailo-Zhuk/Bookhub

---

## Опис

Повноцінна платформа-каталог книг із пошуком, фільтрами та окремими сторінками книг. Користувач може переглядати структурований каталог, читати описи, рекомендації, дивитися відеоогляди з YouTube та переходити на легальні ресурси для придбання або читання книг.

## Функціонал

### Для користувача
- **Каталог книг** — візуальні картки, пошук, фільтри (жанр, теги, мова, формат доступності)
- **Сторінка книги** — обкладинка, назва, автор, опис, відгук, рекомендації
- **YouTube-відеоогляди** — вбудовані або пов'язані відео на сторінці книги
- **Легальні ресурси** — посилання на магазини, електронні сервіси, бібліотеки
- **Навігація** — головна → каталог → сторінка книги → жанри → добірки

### Для адміністратора
- **Адмін панель** — створення, редагування, публікація книг
- **Керування YouTube-посиланнями** — додавання відеооглядів
- **Керування зовнішніми посиланнями** — легальні ресурси для книг

## Структура даних книги

| Поле | Опис |
|------|------|
| Назва | Назва книги |
| Автор | Ім'я автора |
| Обкладинка | Зображення |
| Короткий опис | Стислий опис |
| Розгорнутий опис / відгук | Детальний текст |
| Жанр | Категорія |
| Теги | Ключові слова |
| Мова | Мова книги |
| Рік видання | Рік |
| Обсяг читання | Тривалість або обсяг |
| Рівень складності | Початківець / Середній / Експерт |
| Рекомендація: кому | Цільова аудиторія |
| Рекомендація: коли | Контекст читання |
| YouTube-відео | Посилання на відеоогляд |
| Зовнішні ресурси | Магазини, сервіси, бібліотеки |
| Доступність | Паперова / Електронна / Аудіо |
| Статус публікації | Чернетка / Опубліковано |
